# Phase 4 · Build the Solution

**Entry:** Bedrock facts numbered and locked from Phase 3

## Step 1 · Ask what each fact makes possible

For each numbered fact, ask:

> "Given that [fact N] is true — what does that make possible, or what does it require?"

One fact at a time. The answer becomes raw material for the solution.

## Step 2 · Assemble the solution

Take all the answers from Step 1 and build a concrete solution.

Rules:
- Every claim in the solution must end with a citation: (→ Fact N)
- If a claim cannot point to a fact, remove it or go back and check whether it belongs
- The solution must directly address the problem sentence from Phase 1

## Step 3 · Present and confirm

Present the solution clearly. Then ask:

> "Does this solve the problem we stated in Phase 1?"

Options:
- (a) Yes — this works (Recommended if it addresses the problem sentence directly)
- (b) It solves part of it — [what is missing]
- (c) No — something is wrong

If (b): identify which part is unresolved and trace it back. Is there a belief that was not fully challenged, or a fact that was missed? Return to the relevant phase.

If (c): return to Phase 1 and restate the problem. The original sentence may have captured the wrong thing.

## Exit Condition

User accepts the solution as addressing the confirmed problem statement.

# Phase 2 · List and Question Beliefs

**Entry:** Problem sentence confirmed from Phase 1

## Step 1 · Ask what beliefs surround the problem

Ask the user: what do you believe must be true for this problem to exist the way it does?

Options:
- (a) List every belief freely — anything you assume to be true about this situation
- (b) Answer targeted questions — I will prompt you one at a time
- (c) Both — start freely, then I will ask follow-up questions (Recommended)

## Step 2 · Question each belief

For every belief the user names, ask one of:
- "Why is this true?"
- "What would happen if this were not true?"
- "Has this always been true, or did someone decide it at some point?"

One belief at a time. Wait for the answer before moving to the next.

If the user cannot defend a belief — it is an assumption, not a fact. Mark it as challenged.

If the user can defend it — it is a candidate for Phase 3. Mark it as surviving.

## Step 3 · Check for remaining beliefs

After each belief is addressed, ask:

> "Is there any other belief about this problem we have not questioned yet?"

Options:
- (a) Yes — [name it]
- (b) No, that is all of them (Recommended when nothing comes to mind)

Repeat Step 2 for any new beliefs.

## Exit Condition

User confirms no beliefs remain unchecked.

# Phase 3 · Find Bedrock Facts

**Entry:** All beliefs checked from Phase 2

## Step 1 · Filter what survived

From Phase 2, take every belief marked as "surviving" — ones the user could defend.

For each one, apply this test out loud:

> "Is this still true if we strip away all habit, all industry convention, and all history — and started from zero?"

Options for each item:
- (a) Yes — this is just true, no matter what (keep it as a fact)
- (b) No — it only feels true because of how things have always been done (move it back to challenged)

## Step 2 · Number the facts

Every item that passes the test gets a number. Write them out as a numbered list:

```
1. [fact]
2. [fact]
3. [fact]
```

Plain language. One sentence per fact. No qualifications.

## Step 3 · Check for missing facts

Ask:

> "Is there anything that is just true about this situation that we have not written down yet?"

Options:
- (a) Yes — [state it], then apply the test in Step 1
- (b) No, that is the complete list (Recommended when nothing comes to mind)

## Exit Condition

User confirms the numbered list is complete and nothing on it can be broken down further.
